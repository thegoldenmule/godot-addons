# editor_tool_kit

**Status:** active

## Overview
Editor-only base classes for building in-editor authoring tools in Godot 4.6+ — `EditorToolPlugin`, `ToolService`, `ContentStore`, `EditorToolUi`, the palette/theme, and `BridgeServer`. A tool is a *service + a view*; styling, persistence, status, the enforced header, and optional MCP/CLI access are inherited from the bases. Vendored into consuming projects and self-updating in place from github.com/thegoldenmule/godot-addons.
