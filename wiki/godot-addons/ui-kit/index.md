# ui_kit

**Status:** active

## Overview
Game-agnostic UI shell infrastructure for Godot 4.x — an async stack-FSM router (`UiRouter`), route state (`UiState`), screen scaffolding (`UiScreenScaffold`), control registration (`UiReg`), and a semantic navigation/automation driver (`UiDriver`). Vendored and self-updating from github.com/thegoldenmule/godot-addons; the addon registers no autoloads — the consuming project declares `UiRouter` and `UiDriver`.
